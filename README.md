# JCFC — Squad Decision Lab

A mobile-friendly football squad-building game using real-life player names and original in-game ratings.

You receive a partly completed 4-3-3, a tactical brief, a transfer-credit budget and three options for each empty position. Complete the XI and receive a score out of 10 with a detailed explanation of every decision.

## Play

Once GitHub Pages is enabled for `main` / `root`, the game is available at:

`https://beckburgz1311-ux.github.io/Jcfc/`

## Scoring

The final score combines:

- Tactical fit
- Squad balance
- Decision quality
- Budget management

Player attributes, archetypes and values are bespoke game data for entertainment and are not official football ratings or market valuations.

## Technology

The first version is a dependency-free static web app contained in `index.html`, designed to work on mobile and desktop through GitHub Pages.
